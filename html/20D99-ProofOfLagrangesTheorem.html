proof of Lagrange’s theorem


We know that the cosets H⁢g form a partition of G (see the coset entry for proof of this.) Since G is finite, we know it can be completely decomposed into a finite number of cosets. Call this number n. We denote the ith coset by H⁢ai and write G as

G=H⁢a1∪H⁢a2∪⋯∪H⁢an

since each coset has |H| elements, we have

|G|=|H|⋅n

and so |H| divides |G|, which proves Lagrange’s theorem. □
