Theorem 1 (The Fourth Isomorphism Theorem)

Let G be a group and N⊴G. There is a bijection between G⁢(N), the set of subgroups of G containing N, and the set of subgroups of G/N defined by A→A/N. Moreover, for any two subgroups A,B in G⁢(N), we have

  1. 1.

    A≤B if and only if A/N≤B/N,

  2. 2.

    A≤B implies |B:A|=|B/N:A/N|,

  3. 3.

    ⟨A,B⟩/N=⟨A/N,B/N⟩,

  4. 4.

    (A∩B)/N=(A/N)∩(B/N), and

  5. 5.

    A⊴G if and only if (A/N)⊴(G/N).
