semigroup


A semigroup G is a set together with a binary operation ⋅:G×G⟶G which satisfies the associative property: (a⋅b)⋅c=a⋅(b⋅c) for all a,b,c∈G.

The set G is not required to be nonempty.

Let G,H be two semigroups. A semigroup homomorphism from G to H is a function f:G→H such that f⁢(a⁢b)=f⁢(a)⁢f⁢(b).
