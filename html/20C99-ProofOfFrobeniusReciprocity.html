proof of Frobenius reciprocity


We prove the slightly more general result

Theorem 0.1.

If G is a finite group with subgroup H, α a class function on H and β a class function on G, then

⟨α↑HG,β⟩G=⟨α,β↓HG⟩H

Here we use ↑HG to refer to the induction (http://planetmath.org/InducedRepresentation) to G of a class function on H, and ↓HG to refer to the restriction (http://planetmath.org/RestrictionRepresentation) of a class function on G to one on H.

Proof.
⟨α↑HG,β⟩G=1|G|∑g∈G(1|H|∑t∈Gt-1⁢g⁢t∈Hα(t-1gt))β⁢(g)¯=1|G|⁢|H|∑t∈G(∑g∈Gt-1⁢g⁢t∈Hα(t-1gt))β⁢(g)¯

Since β is a class function, this is the same as

1|G|⁢|H|⁢∑t∈Gg∈Gt-1⁢g⁢t∈Hα⁢(t-1⁢g⁢t)⁢β⁢(t-1⁢g⁢t)¯=1|G|⁢|H|⁢∑h∈H∑t∈Gg∈Gt-1⁢g⁢t=hα⁢(h)⁢β⁢(h)¯

Clearly for every h∈H,t∈G there is a unique g∈G with t-1⁢g⁢t=h, so every element of H is counted |G| times by the sum. Thus the sum is equal to

|G||G|⁢|H|∑h∈Hα(h)β⁢(h)¯=1|H|∑h∈Hα(h)β⁢(h)¯=⟨α,β↓HG⟩H

∎
