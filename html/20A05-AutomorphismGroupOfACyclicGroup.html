automorphism group of a cyclic group


Theorem 1.

The automorphism group of the cyclic group Z/n⁢Z is (Z/n⁢Z)×, which is of order ϕ⁢(n) (here ϕ is the Euler totient function).

Proof.

Choose a generator x for ℤ/n⁢ℤ. If ρ∈Aut⁡(ℤ/n⁢ℤ), then ρ⁢(x)=xa for some integer a (defined up to multiples of n); further, since x generates ℤ/n⁢ℤ, it is clear that a uniquely determines ρ. Write ρa for this automorphism. Since ρa is an automorphism, xa is also a generator, and thus a and n are relatively prime11 If they were not, say (a,n)=d, then (xa)n/d=(xa/d)n=1 so that xa would not generate.. Clearly, then, every a relatively prime to n induces an automorphism. We can therefore define a surjective map

Φ:Aut⁡(ℤ/n⁢ℤ)→(ℤ/n⁢ℤ)×:ρa↦a(modn)

Φ is also obviously injective, so all that remains is to show that it is a group homomorphism. But for every a,b∈(ℤ/n⁢ℤ)×, we have

(ρa∘ρb)⁢(x)=ρa⁢(xb)=(xb)a=xa⁢b=ρa⁢b⁢(x)

and thus

Φ⁢(ρa∘ρb)=Φ⁢(ρa⁢b)=a⁢b(modn)=Φ⁢(ρa)⁢Φ⁢(ρb)

∎
