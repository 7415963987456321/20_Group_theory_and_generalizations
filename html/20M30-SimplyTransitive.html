simply transitive


Let G be a group acting on a set X. The action is said to be simply transitive if it is transitive and ∀x,y∈X there is a unique g∈G such that g.x=y.

Theorem.

A group action is simply transitive if and only if it is free and transitive

Proof.

Necessity follows since g.x=x implies that g=1G because 1G.x=x also. Now assume the action is free and transitive and we have elements g1,g2∈G and x,y∈X such that g1.x=y and g2.x=y. Then g1.x=g2.x⟹g2-1.g1.x=(g2-1⁢g1).x=x hence g2-1⁢g1=1G because the action is free. Thus g1=g2 and so the action is simply transitive. ∎
