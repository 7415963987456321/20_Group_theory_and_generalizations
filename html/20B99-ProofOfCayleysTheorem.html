proof of Cayley’s theorem


Let G be a group, and let SG be the permutation group of the underlying set G. For each g∈G, define ρg:G→G by ρg⁢(h)=g⁢h. Then ρg is invertible with inverse ρg-1, and so is a permutation of the set G.

Define Φ:G→SG by Φ⁢(g)=ρg. Then Φ is a homomorphism, since

(Φ⁢(g⁢h))⁢(x)=ρg⁢h⁢(x)=g⁢h⁢x=ρg⁢(h⁢x)=(ρg∘ρh)⁢(x)=((Φ⁢(g))⁢(Φ⁢(h)))⁢(x)

And Φ is injective, since if Φ⁢(g)=Φ⁢(h) then ρg=ρh, so g⁢x=h⁢x for all x∈X, and so g=h as required.

So Φ is an embedding of G into its own permutation group. If G is finite of order n, then simply numbering the elements of G gives an embedding from G to Sn.
